# Workbench Notes

Use this folder for notes created while testing the framework with the personal AI workbench.

Good uses:

- workbench probe findings
- context packet observations
- integration gaps
- prompts or task goals that worked well
- rough outputs that need human review before being promoted

Do not treat this folder as the source of truth for final knowledge. Promote useful material into `docs/`, `notes/concepts/`, or `projects/`.
