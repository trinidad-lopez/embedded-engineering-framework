# Projects

This folder stores project-level work when a project needs more than a short roadmap entry.

Use the docs for overview and planning:

- `docs/project-roadmap.md`: staged long-term project map.
- `docs/quick-projects.md`: small experiments.
- `docs/progress-tracker.md`: sessions, milestones, and project index.

Use this folder for durable project artifacts:

- project briefs
- design notes
- implementation logs
- debugging notes
- validation notes
- portfolio extraction notes

## Folders

- `planned/`: scoped but not active.
- `active/`: currently being worked.
- `completed/`: finished or archived projects.
