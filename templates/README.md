# Templates

Copy these files when creating new notes, projects, debugging cases, session logs, and portfolio entries.

The same templates are summarized in `docs/templates.md` for quick reading, but these standalone files are easier for editors and the workbench to use directly.

## Files

- `session-log.md`
- `concept-note.md`
- `project-brief.md`
- `debugging-case.md`
- `portfolio-entry.md`
- `weekly-review.md`
